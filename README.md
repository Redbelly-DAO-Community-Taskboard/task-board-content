# task-board-content

Content and community submissions for the Redbelly DAO Community Task Board.

**Tasks covered:** 6 (Redbelly Insights Brand Intro Sequence), 12 (X Account Manager monthly reporting)

---

## Submission Requirements

**Task 6 (Video/Motion Graphics)**
- Final rendered files (1080p MP4 and 4K MP4)
- Open project files (After Effects, Premiere Pro, or Blender)
- Audio stems as separate files
- Audio licence documentation (original composition or licensed assets only)
- YouTube upload link

**Task 12 (X Account Manager)**
- Monthly analytics report (submitted by 5th of following month)
- Report covers follower growth, impressions, engagement rate, link clicks
- Recommendations for next month's content strategy

## Review Criteria

**Task 6:** Institutional aesthetic (appropriate for a financial compliance webinar), no render artefacts, audio licensed, seamless integration into interview footage.

**Task 12:** KPIs met, zero factual errors in any posts, professional tone maintained throughout the month.
